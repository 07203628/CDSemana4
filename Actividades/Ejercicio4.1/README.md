# Ejercicio 4.1: Identificacion de Valores Faltantes

## Objetivo
Detectar valores faltantes y aplicar tecnicas de manejo sobre un DataFrame.

## Entregable
- Script: [ejercicio4_1.py](ejercicio4_1.py)

## Tecnicas aplicadas
1. Deteccion con isnull() y notnull()
2. Conteo de nulos por columna
3. Completitud con info()
4. Manejo con eliminacion de filas
5. Manejo con eliminacion de columnas
6. Imputacion con media
7. Imputacion con mediana
8. Imputacion con moda
