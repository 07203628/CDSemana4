# Ejercicio 4.2: Imputacion de Datos

## Objetivo
Comparar metodos de imputacion y justificar su uso.

## Entregable
- Script: [ejercicio4_2.py](ejercicio4_2.py)

## Metodos incluidos
1. Imputacion por media
2. Imputacion por mediana
3. Imputacion por moda
4. Imputacion hacia adelante (forward fill)
5. Imputacion hacia atras (backward fill)

## Criterio de uso rapido
- Media: datos numericos sin outliers fuertes.
- Mediana: datos numericos con outliers.
- Moda: variables categoricas o discretas.
- Forward/Backward fill: series temporales con continuidad.
