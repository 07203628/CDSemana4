# Ejercicio 16: Tecnicas Avanzadas

## 1) Que es SMOTE para datos desbalanceados
SMOTE (Synthetic Minority Over-sampling Technique) genera ejemplos sinteticos de la clase minoritaria para equilibrar el dataset. No duplica filas exactas: crea nuevos puntos interpolando vecinos cercanos de la clase minoritaria.

Cuando usarlo:
- Clasificacion con fuerte desbalance de clases.
- Cuando el modelo ignora la clase minoritaria.

Precauciones:
- Aplicarlo solo en entrenamiento, nunca antes del split.
- Combinarlo con validacion cruzada y metricas como F1, recall, PR-AUC.

## 2) Que es la imputacion por K-Nearest Neighbors
KNN Imputer estima valores faltantes usando los K registros mas parecidos segun distancia en las demas variables. El faltante se reemplaza con promedio (o esquema similar) de esos vecinos.

Ventajas:
- Usa estructura local de los datos.
- Puede capturar relaciones no lineales simples.

Desventajas:
- Costo computacional mayor.
- Sensible a escala de variables y a outliers.

## 3) Que es Target Encoding
Target Encoding reemplaza categorias por una estadistica del target asociada a cada categoria (por ejemplo, media del target en regresion o tasa positiva en clasificacion).

Ventajas:
- Muy util para variables categoricas de alta cardinalidad.
- Puede mejorar desempeno frente a one-hot en algunos casos.

Riesgo principal:
- Puede causar leakage y sobreajuste si se calcula con todo el dataset.

Buenas practicas:
- Calcular encoding solo con folds de entrenamiento.
- Usar smoothing/regularizacion.
- Validar cuidadosamente fuera de muestra.
